
# Matric-Exam-Paper-Verification-System
The system allows markers, moderators, or educators to upload and review marked exam papers, automatically calculate totals, and detect marking errors such as incorrect additions, missed questions, or inconsistent scoring. This helps improve marking accuracy, reduce human error, and ensure fair results for students.
